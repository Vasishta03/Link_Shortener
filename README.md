
# Link Shortener

This repository contains a simple link shortener application implemented using HTML, CSS, and JavaScript.

### Description

The application allows users to input a URL and generate a shortened version of it. Users can easily copy the shortened URL to their clipboard.

### Features

- Input field for user URL entry
- Button to generate a shortened URL
- Display of the shortened URL with a copy button
- Responsive and user-friendly design

### Installation

Clone this repository to your local machine using:

```bash
git clone https://github.com/Vasishta03/Link_Shortener.git
```
Open the index.html file in your web browser to use the application.

### Usage
Enter the URL you want to shorten in the input field.
Click the "Shorten Link" button.
The shortened URL will be displayed below, with an option to copy it to your clipboard.
